## What to build

Set up Manifest V3 browser extension with Chrome Side Panel API, strict CSP, event-based service worker, Vanilla JS/HTML/CSS (Pure CSS, system fonts), and Readability.js via CDN.

## Acceptance criteria

- [ ] Manifest V3 configured with strict CSP
- [ ] Event-based service worker (not long-running)
- [ ] Chrome Side Panel API integrated (400px default width)
- [ ] Vanilla JS, HTML, Pure CSS (no frameworks)
- [ ] System fonts used throughout
- [ ] Readability.js loaded via CDN
- [ ] Default browser icon used
- [ ] No dark mode, no custom fonts/icons for MVP
